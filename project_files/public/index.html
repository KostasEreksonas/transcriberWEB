Frequently Asked Questions

Whisper is an automatic speech recognition (ASR) system trained on 680,000 hours of multilingual and multitask supervised data collected from the web. OpenAI show that the use of such a large and diverse dataset leads to improved robustness to accents, background noise and technical language. Moreover, it enables transcription in multiple languages, as well as translation from those languages into English. OpenAI is open-sourcing models and inference code to serve as a foundation for building useful applications and for further research on robust speech processing.

An application that implements the Whisper model for transcription and translation is created using Python programming language. A wrapper script for this application is written in PHP. The purpose of this wrapper script is to interact with the user to gather parameters for audio transcription and translation, pass them to the Python app and display the results for the user.

Yes, registration is mandatory in order to use this application.

The registered user provides an URL to a video, specifies parameters for transcription (model, precision, translation, etc.) and presses the button for transcription. Results of the operation will be displayed on the same screen.

As of today, it is planned to add file upload capabilities to transcribe locally uploaded audio files.
